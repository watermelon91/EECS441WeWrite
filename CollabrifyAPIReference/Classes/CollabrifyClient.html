Create a session with a base file and a participant limit.

- (void)createSessionWithBaseFileWithName:(NSString *)sessionName tags:(NSArray *)tags password:(NSString *)password participantLimit:(NSInteger)participantLimit startPaused:(BOOL)startPaused completionHandler:(CreateSessionCompletionHandler)completionHandler

Parameters

sessionName

The name of the session being created. Needs to be unique.

tags

The tags that represent the session being created.

password

A password to protect the session being created. Pass nil if the session does not need a password

participantLimit

Limits the number of participants that a session can have. 0 is the default, indicating no limit

startPaused

Tells the client whether or not to immediately pause all events upon creation.

completionHandler

A completion block that passes back information after a session has been created.

Discussion

Warning: CreateSessionCompletionHanler is called on the main thread
